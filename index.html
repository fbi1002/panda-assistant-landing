入門指南

第一次安裝

  1. 到設定頁設定每日目標與車輛成本。
  2. 填入車輛固定成本與每公里耗材。
  3. 開啟背景 HUD 與定位權限。
  4. 整理常用訊息範本與奧客名單。

待命接單

  1. 在算單頁輸入總運費，切換單筆或疊單模式。
  2. 查看評估卡與每分鐘收益，決定是否接單。
  3. 必要時展開成本與利潤明細或黑名單檢查。

開始配送

  1. 跑單中畫面會顯示站點與本站時間。
  2. 使用送達本站、夾單、找零、里程校正等功能。
  3. 查看任務佇列了解後續站點與預計收入。

背景使用

  1. 退到背景後，HUD 持續顯示即時效益。
  2. 可直接在 HUD 輸入資料（完整版）。

班次結束

  1. 到戰績頁查看今日戰績與月度統計。
  2. 匯出每日存檔為 CSV 或 JSON 以便備份。
