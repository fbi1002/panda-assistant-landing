# 熊貓跑單護

## 專項簡介

「熊貓跑單護」是一款專為外送員打造的跑單效率工具，於以員的配送過程中快速估算每躯訂單的真實收益，並提供流程管理與資料分析功能。本專項包含應用程式 APK 與對應的官方網站，網站採用 GitHub Pages 佈置，方便外送傍伴快速了解功能並下載程式。

## 主要功能

- **接單前預判風險：** 依輸入的運費與單數即時估算每分鐘收益與成本抽解，幫助您判斷是否值得接單。
- **支援複雜情境：** 可切換單筆、疊單與加單三種模式，重新估算整趟配送的效益與時間成本。
- **跑單管理工具：** 在配送中可即時管理站點順序、里線校正、找零與快速新增加單，並顯示動態時薪與淨利/ 內港等數據。
- **背景 HUD 懸浮視窗：** 即使切換至導航或通訊軟體，也能通過懸浮 HUD 零時查看每分鐘收益與站點提醒。
- **戰經與成本設定：** 提供每日與月度戰經統計，並支援主、副車車的油耗/ 電費與固定費設定，讓您計算真實淨利。
- **資料匯出與訊息範本：** 可將戰經匯出為 CSV 或 JSON，以便後續分析，並內建常用訊息範本自動填入訂單編號。

## 安裝方式

1. 前往專項網站並點擊「下載 APK 安裝檔」。
2. 將 APK 儲存至 Android 裝置後開啟，若系統提示阻擋安裝，請前往「設定」並啟用「允許此來源的應用程式」。
3. 安裝完成後，首次開啟程式時請依提示授予定位、背景定位與懸浮視窗權限，以確保功能正常運作。

> **提醒：** 本應用程式非上架於 Google Play，所有資料皆儲存在您的裝置端，請定期備份戰經與設定檔案。

## 使用建議

- 在接單前使用算單評估功能比較每分鐘收益，避免只看總運費而無視時間與成本。
- 配送途中善用伏列管理與 HUD 懸浮視窗，即時掃提站點進度與動態時薪。
- 完成班次後進入戰經頁面查看今日與月度統計，並根據成本設定調整後續接單策略。
